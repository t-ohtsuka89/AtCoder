## 準備
.env.exampleを参考に、AtCoderアカウントのログイン情報を記述した.envファイルをプロジェクトのルートディレクトリに作成しておいてください。
## タスク
現在開いているファイルを対象にタスクが実行されます。
### テスト
F1 -> Run task -> test
### 提出
F1 -> Run task -> submit
